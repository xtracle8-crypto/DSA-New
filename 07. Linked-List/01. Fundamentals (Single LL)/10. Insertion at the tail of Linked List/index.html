Insertion at the tail of Linked List

Easy dsa
Linked List

Problem Statement

Given the head of a singly linked list and an integer X, insert a node with value X at the tail of the linked list and return the head of the modified list.

Constraints

  • 0 <= number of nodes in the Linked List <= 1000
  • 0 <= ListNode.val <= 100
  • 0 <= X <= 100

Example 1

Input: linkedList = [1, 2, 3], X = 7

Output: [1, 2, 3, 7]

Explanation:

7 was added as the last node.

Example 2

Input: linkedList = [], X = 0

Output: [0]

Explanation:

0 was added as the last/only node.

Example 3

Input: linkedList = [5, 6], X = 8

Editorial: Solution

Intuition:

Traverse the list to the last node, then set it's next to the new node. If the list is empty, set the head to the new node.

Approach:

  • Create a new node with the given value and set its next to NULL.
  • Initialize a temporary pointer to the head to traverse the linked list.
  • Traverse the list using the temporary pointer until you reach the last node.
  • Point the next of the last node to the newly created node, making it the new tail of the linked list.
  • Return the head of the linked list to get the updated linked list.

using namespace std;

// Definition of singly linked list
struct ListNode
{
    int val;
    ListNode *next;
    ListNode(int data1)
    {
        val = data1;
        next = NULL;
    }
    ListNode(int data1, ListNode *next1)
    {
        val = data1;
        next = next1;
    }
};

// Solution class
class Solution {
public:
    // Function to insert a new node at the tail of the linked list
    ListNode* insertAtTail(ListNode* &head, int X) {
        if (head == NULL)
            return new ListNode(X);

        ListNode* temp = head;
        // Traversing until the last node
        while (temp->next != NULL) {
            temp = temp->next;
        }
        
        ListNode* newNode = new ListNode(X);
        temp->next = newNode;

        return head;
    }
};

// Helper Function to print the linked list
void printLL(ListNode* head) {
    while (head != NULL) {
        cout << head->val << " ";
        head = head->next;
    }
    cout << endl;
}

int main() {
    // Create a linked list from a vector
    vector arr = {10, 20, 30};
    int val = 40;
    ListNode* head = new ListNode(arr[0]);
    head->next = new ListNode(arr[1]);
    head->next->next = new ListNode(arr[2]);
    
    // Print the original list
    cout << "Original List: ";
    printLL(head);
    
    // Create a Solution object 
    Solution sol;
    head = sol.insertAtTail(head, val);

    // Print the modified linked list
    cout << "List after inserting the given value at the tail:";
    printLL(head);

    return 0;
}

package main

import "fmt"

// ListNode defines a node in the singly linked list
type ListNode struct {
    Data  int
    Next *ListNode
}

// InsertAtTail inserts a new node with value X at the tail of the list
func InsertAtTail(head *ListNode, X int) *ListNode {
    // If the list is empty, create and return a new node
    if head == nil {
        return &ListNode{Data: X}
    }

    temp := head
    // Traversing until the last node
    for temp.Next != nil {
        temp = temp.Next
    }
    // Create the new tail node and link it
    temp.Next = &ListNode{Data: X}

    return head
}

// printLL prints the linked list values
func printLL(head *ListNode) {
    for head != nil {
        fmt.Printf("%d ", head.Data)
        head = head.Next
    }
    fmt.Println()
}

func main() {
    // Create a linked list from a slice: 10 -> 20 -> 30
    head := &ListNode{Data: 10}
    head.Next = &ListNode{Data: 20}
    head.Next.Next = &ListNode{Data: 30}

    val := 40
    fmt.Print("Original List: ")
    printLL(head)

    // Insert value at the tail
    head = InsertAtTail(head, val)

    fmt.Print("List after inserting the given value at the tail: ")
    printLL(head)
}

Complexity Analysis:

Time Complexity: O(N) for traversing the linked list and inserting the new node after the tail. Here N is the length of the Linked List.

Space Complexity: O(1) as no extra space used.
