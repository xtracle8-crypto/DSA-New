Delete Tail of Doubly Linked List

Easy dsa
Doubly-Linked ListHash TableLinked List

Problem Statement

Given the head of a doubly linked list, remove the node at the tail of the linked list and return the head of the modified list.


The tail is the last node of the linked list.

Constraints

  • n == Number of nodes in the linked list
  • 1 <= n <= 100
  • 0 <= ListNode.val <= 100

Example 1

Input: head = [1, 2, 3]

Output: head = [1, 2]

Explanation:

  • The node with value 3 was removed.

Example 2

Input: head = [7]

Output: head = [ ]

Explanation:

  • Note that the head has null value after the removal.

Example 3

Input: head = [2, 4]

Editorial: Solution

Intuition:

Update the linkage between its last node and its second last node. As a doubly linked list is bidirectional, set the second last node's next pointer and the last node's back pointer to null. Then,return the head as the result.

Approach:

  • Start at the head of the list.
  • Iterate through the list until the last node (tail) is reached.
  • Find the second-to-last node (prev) using the tail's back pointer.
  • Set the next pointer of the prev node to null.
  • Set the back pointer of the tail node to null.
  • Return the head of the list.

Note: In C++, explicitly delete the last node to free memory. In Java, garbage collection handles this.

{
    int val;

// Solution class
class Solution {
public:
    // Function to delete the tail of a doubly linked list 
    ListNode* deleteTail(ListNode* head) {
        if (head == nullptr || head->next == nullptr) {
            return nullptr;  
        }
        
        // Navigate to the tail of the linked list
        ListNode* tail = head;
        while (tail->next != nullptr) 
            tail = tail->next; 
        
        // Update the pointers
        ListNode* newTail = tail->prev;
        newTail->next = nullptr;
        tail->prev = nullptr;
        
        // Free memory 
        delete tail;  
        
        return head; // Return head of modified list
    }
};

ListNode* arrayToLinkedList(vector &nums) {
    // If array is empty, return nullptr
    if (nums.empty()) return nullptr; 

    // Create head node with first element of the array
    ListNode* head = new ListNode(nums[0]); 
    // Initialize 'prev' to the head node
    ListNode* prev = head;             

    for (int i=1; i < nums.size(); i++) {
        // Create a new node 
        ListNode* temp = new ListNode(nums[i], nullptr, prev);
        // Update 'next' pointer
        prev->next = temp;    
        // Move 'prev' to newly created node
        prev = temp;         
    }
    
    // Return head
    return head;  
}

package main

import "fmt"

/*
Definition of doubly linked list:
*/
type ListNode struct {
	Val  int
	Next *ListNode
	Prev *ListNode
}

// DeleteTail deletes the tail node from a doubly linked list and returns the modified head.
func DeleteTail(head *ListNode) *ListNode {
	if head == nil || head.Next == nil {
		return nil
	}

	// Navigate to the tail of the linked list
	tail := head
	for tail.Next != nil {
		tail = tail.Next
	}

	// Update the pointers
	newTail := tail.Prev
	newTail.Next = nil
	tail.Prev = nil // Disconnect the old tail

	return head // Return head of modified list
}

// arrayToLinkedList converts an array to a doubly linked list.
func arrayToLinkedList(nums []int) *ListNode {
	if len(nums) == 0 {
		return nil
	}

	// Create head node
	head := &ListNode{Val: nums[0]}
	prev := head

	for i := 1; i < len(nums); i++ {
		node := &ListNode{Val: nums[i], Prev: prev}
		prev.Next = node
		prev = node
	}

	return head
}

// printLL prints the linked list.
func printLL(head *ListNode) {
	for head != nil {
		fmt.Print(head.Val, " ")
		head = head.Next
	}
	fmt.Println()
}

func main() {
	nums := []int{1, 2, 3, 4, 5}

	// Create doubly linked list
	head := arrayToLinkedList(nums)

	// Print original list
	fmt.Print("Original List: ")
	printLL(head)

	// Delete tail
	head = DeleteTail(head)

	// Print modified list
	fmt.Print("Modified List: ")
	printLL(head)
}

Complexity Analysis:

Time Complexity: O(N), where N is the number of nodes in the DLL.
This is because we traverse the entire DLL to reach the end to delete the tail.

Space Complexity: O(1) as no extra space used.
