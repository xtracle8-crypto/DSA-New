Removing given node in Doubly Linked List

Easy dsa
Doubly-Linked ListLinked List

Problem Statement

Given a node's reference within a doubly linked list, remove that node from the linked list while preserving the list's integrity.


You will only be given the node's reference, not the head of the list. It is guaranteed that the given node will not be the head of the list.

Constraints

  • 2 <= Number of nodes in the list <= 100
  • 0 <= ListNode.val <= 100
  • Node is guaranteed to be a part of the linked list and will not be the head

Example 1

Input: head = [1, 3, 5], node = 3

Output: [1, 5]

Explanation:

  • The referenced node with value 3 was removed.

Example 2

Input: head = [1, 3, 7], node = 7

Output: head = [1, 3]

Explanation:

  • The referenced node with value 7 was removed.

Example 3

Input: head = [1, 5], node = 5

Editorial: Solution

Intuition:

Deleting a node from a doubly linked list involves locating the node and adjusting the pointers of its adjacent nodes to bypass it, ensuring the previous node points to the next node and vice versa. Edge cases for the head and tail nodes should be handled separately, and the memory of the deleted node should be freed if required. This process maintains the list's integrity while removing the node.

Approach:

  • Identify the previous and next nodes of the node to be deleted using its back and next pointers.
  • Update the next pointer of the previous node to point to the next node, and update the back pointer of the next node to point to the previous node.
  • Set the next and back pointers of the node to be deleted to null to fully disconnect it from the list.
  • Delete the node (C++ only). In C++, explicitly delete the node to free memory. In Java, memory management is automatic and handled by the garbage collector.

// Solution class
class Solution {
public:
    /* Function to delete the given node 
    from doubly linked list */
    void deleteGivenNode(ListNode* node) {
        ListNode* prev = node->prev;
        ListNode* front = node->next;

        // Edge case if the given node is the tail node
        if (front == nullptr) {
            prev->next = nullptr;
            node->prev = nullptr;
            delete node;
            return;
        }

        // Disconnect node 
        prev->next = front;
        front->prev = prev;

        // Set node's pointers to NULL
        node->next = nullptr;
        node->prev = nullptr;

        // Free memory
        delete node;
    }
};

// Definition of doubly linked list
type ListNode struct {
	Data int
	Next *ListNode
	Prev *ListNode
}

/* Function to delete the given node
from doubly linked list */
func deleteGivenNode(node *ListNode) {
	prev := node.Prev
	front := node.Next

	// Edge case if the given node is the tail node
	if front == nil {
		prev.Next = nil
		node.Prev = nil
		return
	}

	// Disconnect node
	prev.Next = front
	front.Prev = prev

	// Set node's pointers to NULL
	node.Next = nil
	node.Prev = nil
}

// Helper Function to convert an array to a doubly linked list
func arrayToLinkedList(nums []int) *ListNode {
	// If array is empty, return nil
	if len(nums) == 0 {
		return nil
	}

	// Create head node with first element of the array
	head := &ListNode{Data: nums[0]}
	// Initialize 'prev' to the head node
	prev := head

	for i := 1; i < len(nums); i++ {
		// Create a new node
		temp := &ListNode{Data: nums[i], Prev: prev, Next: nil}
		// Update 'next' pointer
		prev.Next = temp
		// Move 'prev' to newly created node
		prev = temp
	}

	// Return head
	return head
}

func printLL(head *ListNode) {
	for head != nil {
		fmt.Print(head.Data, " ")
		head = head.Next
	}
	fmt.Println()
}

	nums := []int{1, 2, 3, 4, 5}

	head := arrayToLinkedList(nums)
	// Node to be deleted

	// Print the Original list
	fmt.Print("Original List: ")
	printLL(head)

	/* Function call to delete the given
	node from the doubly linked list */
	deleteGivenNode(node)

	// Print the Modified list

}
