Insert before given node in Doubly Linked List

Easy dsa
Doubly-Linked ListLinked List

Problem Statement

Given a node's reference within a doubly linked list and an integer X, insert a node with value X before the given node in the linked list while preserving the list's integrity.


You will only be given the node's reference, not the head of the list. It is guaranteed that the given node will not be the head of the list.

Constraints

  • n == Number of nodes in the Linked List
  • 2 <= n <= 100
  • 0 <= ListNode.val <= 100
  • 0 <= X <= 100
  • It is guaranteed the given node will be a part of a doubly linked list and will not be its head.

Example 1

Input: head = [1, 2, 6], node = 6, X = 7

Output: head = [1, 2, 7, 6]

Explanation: Note that the head was not given to the function.

Example 2

Input: head = [7, 5, 15], node = 5, X = 10

Output: head = [7, 10, 5, 15]

Explanation: The node with value 5 was referenced, thus the new node was added before the given node.

Example 3

Input: head = [7, 6, 5], node = 5, X = 10

Editorial: Solution

Intuition:

Identify the preceding node, create a new node and insert it between the identified node and its successor, then readjust links.

Approach:

  • Track the node to which the back pointer of the referenced node is pointing, and call it the previous node.
  • Create a new node with its data as the given value, its back pointer as the previous node, and its next pointer as the referenced node.
  • Update the next pointer of the previous node to point to the newly created node.

class ListNode {
public:
    int data;
    ListNode *prev;
    ListNode *next;
    ListNode() : data(0), prev(nullptr), next(nullptr) {}
    ListNode(int x) : data(x), prev(nullptr), next(nullptr) {}

// Solution class
class Solution {
public:
    /* Function to insert a new node before
    given node in a doubly linked list */
    void insertBeforeGivenNode(ListNode* node, int X) {
        // Get node before the given node
        ListNode* prev = node->prev;
        
        // Create new node 
        ListNode* newNode = new ListNode(X, prev, node);
        
        // Connect newNode
        prev->next = newNode;
        node->prev = newNode;
        
        // void function to just update
        return;
    }
};

// Helper Function to convert an array to a doubly linked list
ListNode* arrayToLinkedList(vector &nums) {
    // If array is empty, return nullptr
    if (nums.empty()) return nullptr; 

    // Create head node with first element of the array
    ListNode* head = new ListNode(nums[0]); 
    // Initialize 'prev' to the head node
    ListNode* prev = head;             

    for (int i=1; i < nums.size(); i++) {
        // Create a new node

        // Update 'next' pointer
        prev->next = temp;    
        // Move 'prev' to newly created node
        prev = temp;         
    }
    
    // Return head
    return head;  
}

package main

import "fmt"

// Definition of doubly linked list
type ListNode struct {
	Data int
	Next *ListNode
	Prev *ListNode
}

/* Function to insert a new node before
    given node in a doubly linked list */
func InsertBeforeGivenNode(node *ListNode, X int) {
    if node == nil || node.Prev == nil {
        // Can't insert before head or nil node
        return
    }

    // Get node before the given node
    prev := node.Prev

    // Create new node 
    newNode := &ListNode{Data: X, Next: node, Prev: prev}

    // Connect newNode
    prev.Next = newNode
    node.Prev = newNode
}

// arrayToLinkedList builds a doubly linked list from a slice
func arrayToLinkedList(nums []int) *ListNode {
    if len(nums) == 0 {
        return nil
    }
    head := &ListNode{Data: nums[0]}
    prev := head
    for _, v := range nums[1:] {
        node := &ListNode{Data: v, Prev: prev}
        prev.Next = node
        prev = node
    }
    return head
}

// printLL prints all values in the list
func printLL(head *ListNode) {
    for curr := head; curr != nil; curr = curr.Next {
        fmt.Printf("%d ", curr.Data)
    }
    fmt.Println()
}

func main() {
    nums := []int{1, 2, 4, 5}
    head := arrayToLinkedList(nums)

    // Identify the node before which to insert
    node := head.Next.Next // points to the 3rd node (value 4)

    fmt.Print("Original List: ")
    printLL(head)

    // Perform insertion
    InsertBeforeGivenNode(node, 3)

    fmt.Print("Modified list: ")
    printLL(head)
}

Time Complexity: O(1) because only a constant number of pointer updates are being performed regardless of the size of the Doubly Linked List.

Space Complexity: O(1) as no extra space is used.
