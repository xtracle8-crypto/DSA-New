Insert node before (kth node) in Doubly Linked List

Easy dsa
Doubly-Linked ListLinked List

Problem Statement

Given the head of a doubly linked list and two integers X and K, insert a new node with value X, before the Kth node of the linked list and return the head of the modified linked list.

Constraints

  • n == Number of nodes in the linked list
  • 1 <= n <= 100
  • 0 <= ListNode.val <= 100
  • 0 <= X <= 100
  • 1 <= K <= n

Example 1

Input: head = [1, 3, 5], X = 7, K = 2

Output: head = [1, 7, 3, 5]

Explanation: A node with value 7 was added before the 2nd node.

Example 2

Input: head = [5], X = 7, K = 1

Output: head = [7, 5]

Explanation: A node with value 7 was added, note that the head was changed.

Example 3

Input: head = [4, 5], X = 10, K = 2

Editorial: Solution

Intuition:

Inserting a node in front of the Kth node in a doubly linked list involves first locating the Kth node by counting up to the Kth position. Then, adjust the pointers of the adjacent nodes to insert the new node before the Kth node.

Approach:

  • Traverse the doubly linked list until reaching the Kth node, keeping a count. When the count reaches K, stop the traversal. Use a temporary pointer to mark the Kth node.
  • Identify the node before the Kth node and prepare a new node with the required data, setting its previous and next pointers to the appropriate nodes.
  • Update the next pointer of the node before the Kth node to point to the new node, and update the back pointer of the Kth node to point to the new node.

// Solution class
class Solution {
public:
    /* Function to insert a node before the
    Kth node in a doubly linked list */
    ListNode* insertBeforeKthPosition(ListNode* head, int X, int K) {
        // If node has to be inserted before the head
        if (K == 1) {
            ListNode* newHead = new ListNode(X, nullptr, head);
            head->prev = newHead;
            return newHead;
        }

        // Temporary pointer 
        ListNode* temp = head;

        // Reach kth node
        int count = 0;
        while (temp != nullptr) {
            count++;
            
            // If kth node is reached, Break out of the loop
            if (count == K) break;
            
            // Otherwise Keep moving temp forward
            temp = temp->next;
        }
        
        // Track the node 
        ListNode* prev = temp->prev;

        // Create new node with data as X
        ListNode* newNode = new ListNode(X, prev, temp);

        // Join new node 
        prev->next = newNode;
        temp->prev = newNode;

        // Return head 
        return head;
    }
};

package main

import "fmt"

// Definition of doubly linked list
type ListNode struct {
	Data int
	Next *ListNode
	Prev *ListNode
}

// InsertBeforeKthPosition inserts a node with value X before the K-th node in a doubly linked list
func InsertBeforeKthPosition(head *ListNode, X int, K int) *ListNode {
    // If inserting before head (K == 1)
    if K == 1 {
        newHead := &ListNode{Data: X, Next: head, Prev: nil}
        if head != nil {
            head.Prev = newHead
        }
        return newHead
    }

    // Traverse to find the K-th node
    curr := head
    count := 1
    for curr != nil && count < K {
        curr = curr.Next
        count++
    }
    // If K is out of range, do nothing
    if curr == nil {
        return head
    }

    prev := curr.Prev
    newNode := &ListNode{Data: X, Next: curr, Prev: prev}
    if prev != nil {
        prev.Next = newNode
    }
    curr.Prev = newNode

    return head
}

// arrayToLinkedList builds a doubly linked list from a slice
func arrayToLinkedList(nums []int) *ListNode {
    if len(nums) == 0 {
        return nil
    }
    head := &ListNode{Data: nums[0]}
    prev := head
    for _, v := range nums[1:] {
        node := &ListNode{Data: v, Prev: prev}
        prev.Next = node
        prev = node
    }
    return head
}

// printLL prints the values of the doubly linked list
func printLL(head *ListNode) {
    for current := head; current != nil; current = current.Next {
        fmt.Printf("%d ", current.Data)
    }
    fmt.Println()
}

func main() {
    nums := []int{1, 2, 3, 5}
    head := arrayToLinkedList(nums)

    fmt.Print("Original List: ")
    printLL(head)

    head = InsertBeforeKthPosition(head, 4, 4)

    fmt.Print("Modified list: ")
    printLL(head)
}

Complexity Analysis:

Time Complexity: O(N), where N is the number of nodes in the Linked List. In the worst case, it involves traversing N nodes in the Doubly Linked List to reach the last element. In the best case, when K is 0 (insertion at the head), the time complexity is O(1) as it involves a constant number of operations. In the average case, it's O(K).

Space Complexity: O(1) as no extra space is used.
