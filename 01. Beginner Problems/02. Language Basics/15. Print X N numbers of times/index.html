Print X N numbers of times

Easy dsa
ImplementationSimulation

Problem Statement

Given two integers X and N, print the value X on the screen N times.

  • Separate each number by a single space.
  • Do not add a space after the last number.
  • After printing all N numbers, move to the next line.
  • If N = 0, still move to the next line (print an empty line).

Constraints

  • -100 <= X <= 100
  • 0 <= N <= 100

Example 1

Input: X = 7, N = 5

Output: 7 7 7 7 7

Example 2

Input: X = 15, N = 1

Output: 15

Example 3

Input: X = -5, N = 4

Editorial: Solution

Approach:

  • Receive Inputs: Capture the integers X and N from the user.
  • Use a loop to print the value X exactly N times, ensuring a space separates each value.

    #include 
using namespace std;

class Solution {
public:
    // Function to print the value X on the screen N times
    void printX(int X, int N) {
        for (int i = 0; i < N; ++i) {
            cout << X;

            if (i < N - 1) {
                cout << " ";
            }
        }

        cout << endl;
    }
};

int main() {
    // Creating an instance of Solution class 
    Solution sol;
    int X = 7, N = 5;
    
    // Function call to print the value X, N times
    sol.printX(X, N);
    return 0;
}
