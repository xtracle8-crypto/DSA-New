If Else Adult Teen Problem

Easy dsa
Implementation

Problem Statement

Given an integer age, print on the screen:

  • Adult if age >= 18
  • Teen if age < 18

Do not change the case of any letter in "Adult" and "Teen" while printing the answer.


For printing use:-

  • for C++ : cout << variable_name<
  • for Java : System.out.println();
  • for Python : print()
  • for Javascript : console.log()

Constraints

  • 0 <= age <= 100

Example 1

Input: age = 19

Output: Adult

Explanation: age is greater than or equal to 18.

Example 2

Input: age = 7

Output: Teen

Explanation: age is less than 18.

Example 3

Input: age = 18

Editorial: Solution

Approach:

  • Receive Input: Capture the integer input age from the user.
  • If age is greater than or equal to 18, print "Adult".
  • Otherwise, print "Teen".

    #include 
using namespace std;

class Solution {
public:

    /* Function to check if the person
    in an adult or a teen */
    void isAdult(int age) {
        
       // If age is greater than 18
        if (age >= 18) {
            // The person is an adult
            cout << "Adult\n";
        } 
        
        // Otherwise
        else {
            // The person is a teen
            cout << "Teen\n";
        }
    }
};

int main() {
    // Creating an instance of Solution class
    Solution solution;

    //Take age as input
    int age;
    cout << "Enter your age: ";
    cin >> age;

    /* Function call to check if the person
    in an adult or a teen */
    solution.isAdult(age);
    return 0;
}

        fmt.Println("Teen")

}

}

    Time Complexity: O(1), The operations of receiving input and printing output are executed once regardless of the size of the input.
