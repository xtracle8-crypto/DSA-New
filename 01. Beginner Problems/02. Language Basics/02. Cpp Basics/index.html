Cpp Basics

(The Language of Speed)

Welcome to the Coding World!
C++ (CPP) is a powerful, general-purpose programming language widely used for system development, software engineering, and competitive programming. In this article, we’ll explore the basics of C++, including what it is, its core syntax, and how a simple program works behind the scenes. By understanding these fundamentals, you’ll build a strong foundation for writing efficient and structured C++ code.

1. Basic Skeleton

Every human has a skeleton (bones) that gives them structure. Similarly, every C++ program has a standard skeleton.

Here is the minimum code required to run a C++ program:

1234567
#include 
using namespace std;

int main() {
    // Write your code here
    return 0;
}

Understanding the Parts:

  • #include : This tells the computer: "Please include tools for Input and Output". Without this, we cannot take input or print anything. If we require any other tools (say maths tool), we include its corresponding header file by writing #include .
  • using namespace std;: This is a shortcut. C++ organizes its standard tools (like the one used for printing) inside a container called std. This line lets us use those tools directly without having to prefix them with std:: every time.
  • int main(): This is the starting point. When we run a program, the computer looks for main() and starts executing from there.
  • { ... }: These curly braces mark the beginning and end of the main function. Everything inside gets executed sequentially.
  • return 0;: This tells the computer: "The program finished successfully."

2. Data Types

In real life, we have different types of containers for different things:

  • A bottle for water (Liquid)
  • A box for shoes (Solid)
  • A wallet for money (Cash)

In C++, we have Data Types to store different kinds of data. These act as containers (variables).

Common Data Types:

  • int (Integer): Stores whole numbers.
    Example: 5, -10, 100
  • float (Floating Point): Stores decimal numbers.
    Example: 3.14, 2.5, -0.1
  • char (Character): Stores a single letter or symbol.
    Example: 'a', 'Z', '$'
  • bool (Boolean): Stores only True or False.
    Example: true, false
  • string (String): Stores text (collection of characters).
    Example: "Hello", "C++"

Code Example:

1234567891011
#include 
using namespace std;

int main() {
    int age = 25;
    float height = 5.9;
    char grade = 'A';
    bool isStudent = true;
    string name = "Rahul";
    return 0;
}


3. Input / Output (I/O)

How do we talk to the computer?

  • Output: Telling the computer to show something on screen. We use cout (See-Out).
  • Input: Asking the user to type something. We use cin (See-In).

Code Example:

1234567891011121314151617
#include 
using namespace std;

int main() {
    int age;

    // Output: Ask user a question
    cout << "Enter your age: ";
    
    // Input: Read what user types and store in 'age'
    cin >> age;
    
    // Output: Show the result
    cout << "Your age is: " << age;
    
    return 0;
}

Note the arrows:
cout << (Arrows go OUT towards the screen)
cin >> (Arrows go IN towards the variable)

4. If Else (Decision Making)

Life is full of decisions: "If it rains, I will take an umbrella. Else, I will not."

In C++, we use if and else for this.

Code Example:

1234567891011121314
#include 
using namespace std;

int main() {
    int money = 500;

    if (money >= 1000) {
        cout << "I will buy a pizza";
    } else {
        cout << "I will buy a burger";
    }
    
    return 0;
}

Explanation: The program checks if money is greater or equal to 1000. Since 500 is not, it goes to the else part and prints "I will buy a burger".

5. For Loop (Repetition)

Imagine your teacher tells you: "Write 'I am sorry' 100 times."

You wouldn't want to write the same line of code 100 times. Instead, we use a Loop.

Code Example:

1234567891011
#include 
using namespace std;

int main() {
    // Print "Hello" 5 times
    for (int i = 1; i <= 5; i++) {
        cout << "Hello" << endl;
    }
    
    return 0;
}

Breakdown:
The syntax is written as: for (initialization; condition; updation)

  • Initialization (int i = 1): This is where we start. It runs only once at the beginning. (This part can be left empty).
  • Condition (i <= 5): The computer checks this before every round. If true, the loop continues. (This part can be left empty).
  • Updation (i++): This happens at the end of every round, usually to update the counter. (This part can be left empty).

6. While Loop

A while loop is like a for loop, but simpler. It only cares about the condition.

"While you are hungry, eat."

Code Example:

12345678910111213
#include 
using namespace std;

int main() {
    int count = 1;

    while (count <= 5) {
        cout << count << " ";
        count++; // Important! Increase count, otherwise it runs forever.
    }
    
    return 0;
}

Output: 1 2 3 4 5

7. Switch Case

When you have many options to choose from (like a menu), using many if-else checks gets messy. We use switch instead.

Code Example:

12345678910111213141516171819202122
#include 
using namespace std;

int main() {
    int day = 3;

    switch (day) {
        case 1:
            cout << "Monday";
            break;
        case 2:
            cout << "Tuesday";
            break;
        case 3:
            cout << "Wednesday";
            break;
        default:
            cout << "Invalid day";
    }
    
    return 0;
}

Explanation: The computer checks day. It matches with case 3 and prints "Wednesday".

  • The break statement: It is used to "break out" of the switch block. If you omit break, the program continues executing the code in the next cases regardless of whether they match (this is called fall-through). We add it to ensure only the matching block runs.
  • The default keyword: This specifies the code to run if none of the cases match the value. It acts like the final else in an if-else chain.

8. Arrays

What if we need to store marks of 50 students? Creating 50 variables (m1, m2, m3...) is bad.

An Array is a single variable that can hold multiple values of the same type. Think of it as an egg tray.

Code Example:

123456789101112131415
#include 
using namespace std;

int main() {
    // Create an array of 5 integers
    int marks[5] = {90, 85, 92, 78, 88};
    
    // Accessing the first student's marks
    cout << marks[0] << endl; // Output: 90
    
    // Accessing the third student's marks
    cout << marks[2] << endl; // Output: 92
    
    return 0;
}

Important: In C++, counting(indexing) starts from 0. So the first element is at index 0.

9. Strings

A string is used to store text. It is actually a chain of characters.

Code Example:

10. Functions

A function is a block of code that does a specific task. We write it once and call it whenever we need it.

It's like a "helper" or a "servant". We teach the servant how to make tea once, and then just order "Make Tea" anytime.

Code Example:

12345678910111213
#include 
using namespace std;

// Function Definition
void sayHello() {
    cout << "Hello user!" << endl;
}

int main() {
    sayHello(); // Calling the function
    sayHello(); // Calling it again
    return 0;
}

Functions keep our code clean and reusable.

11. Pass by Value vs Pass by Reference

This is a slightly advanced but important concept. When we pass variables to functions, how are they sent?

Pass by Value (The Photocopy)

In Pass By Value, the function gets a copy of the variable. Changing the copy does NOT change the original.

123456789101112
#include 
using namespace std;

void changeValue(int x) {
    x = 100; // Changes only the copy
}

int main() {
    int num = 10;
    changeValue(num);
    cout << num; // Output: 10 (Original didn't change!)
}

Pass by Reference (The Original)

In Pass by Reference, we send the address of the variable (using &). The function works on the original variable.

123456789101112
#include 
using namespace std;

void changeRealValue(int &x) { // Note the '&'
    x = 100; // Changes the actual variable
}

int main() {
    int num = 10;
    changeRealValue(num);
    cout << num; // Output: 100 (It changed!)
}

Analogy:
- Pass by Value: Sharing a photo of your homework. If your friend writes on the photo, your notebook stays clean.
- Pass by Reference: Giving your actual notebook. If your friend writes in it, your notebook is changed.
