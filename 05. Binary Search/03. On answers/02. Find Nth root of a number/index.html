Find Nth root of a number

Medium dsa
ArrayBinary Search

Problem Statement

Given two numbers N and M, find the Nth root of M. The Nth root of a number M is defined as a number X such that when X is raised to the power of N, it equals M. If the Nth root is not an integer, return -1.

Constraints

  •   1 <= N <= 30
  •   1 <= M <= 109

Example 1

Input: N = 3, M = 27

Output: 3

Explanation: The cube root of 27 is equal to 3.

Example 2

Input: N = 4, M = 69

Output:-1

Explanation: The 4th root of 69 does not exist. So, the answer is -1.

Example 3

Input: N = 4, M = 81

Fun Fact

This concept is widely used in game development, 3D graphics and image manipulation software. It's a key part in calculating distances and generating effects such as fog or light falloff. Additionally, the problem comes into play in audio processing for generating specific tones or adjusting volume and pitch. This mathematical operation, finding the Nth root, is an integral part of many algorithmic solutions in software.

Editorial: Linear Search

Intuition:

Perform a simple linear search in range [1,M]. Calculate the value of x raised to the power N for every number x in this range. If it is equal to the given number then, x is the Nth root of the number. If no such number(x) exists, return -1 as an answer.

Approach:

Working of nthRoot(N, M):

  • Iterate from 1 to M(given number) using a for loop. Calculate the value of x raised to the power n as follows:
    • Start with the result as 1 and keep track of the base value.
    • If the exponent is odd, multiply the result by the base and reduce the exponent by one.
    • If the exponent is even, square the base and half the exponent.
    • Continue the process until the exponent becomes zero, and the final result is returned.
  • If the result is equal to M, this means x is the Nth root of M. So, return x from this step.
  • If the calculated value is smaller than M, then continue to next iteration.
  • If the calculated value is greater than M, this means we have got a bigger number than our answer and until now we have not found any number that can be our answer. In this case, our answer does not exist and break out from this step and return -1.

class Solution {

    exponentiation by squaring method*/
    long long Pow(int b, int exp) {
        long long ans = 1;
        long long base = b;
        
        // Exponentiation by squaring method
        while (exp > 0) {
            if (exp % 2) {
                exp--;
                ans = ans * base;
            } else {
                exp /= 2;
                base = base * base;
            }
        }
        return ans;
    }
public:
    /* Function to find the nth root
    of m using linear search*/

            long long val = Pow(i, N);
            
            /* Check if the computed
            value is equal to M */

                // Return the root value
                return i; 
            } else if (val > M * 1ll) {
                break;
            }
        }
        // Return -1 if no root found
        return -1;

};

Complexity Analysis:

Time Complexity: O(N*logN)
The for loop runs takes O(M) time and calculating Pow(x, N) takes O(logN) time. However, since the for loop breaks as soon as the result of Pow(x, N) becomes greater than the M, thus, the for loops actually runs only for N iterations making overall complexity as O(N*logN).

Space Complexity: O(1), as there are only a couple of variables used.

Editorial: Binary Search

Intuition:

The idea is to use binary search to efficiently find the Nth root of M. Although binary search is typically applied on sorted arrays, it can also be used on a monotonic search space.

In this problem, the possible values of the Nth root lie in the range [1, M]. As the value of xN increases monotonically with x, binary search can be safely applied on this range.

Approach:

Why a helper function is needed:
Directly computing midN may cause integer overflow when N and M are large. To avoid this, we use a helper function that computes the power using Exponentiation by Squaring and stops early if the value exceeds M.

Working of NthRoot(N, M):
  1. Initialize two pointers: low = 1 and high = M.
  2. Run a binary search loop while low ≤ high.
  3. Compute mid = (low + high) / 2.
  4. Call the helper function to compare midN with M:
    1. If it returns 1: midN == M, so mid is the Nth root. Return it.
    2. If it returns 0: midN < M, so search the right half (low = mid + 1).
    3. If it returns 2: midN > M, so search the left half (high = mid - 1).
  5. If the loop ends without finding a valid root, return -1.
Working of helper function func(mid, N, M):
  • The function computes midN using Exponentiation by Squaring, which reduces time complexity from O(N) to O(log N).
  • If N is even, the base is squared and N is halved.
  • If N is odd, the base is multiplied once and N is reduced by 1.
  • At any point, if the computed value exceeds M, the function immediately returns 2 (early stopping).
  • After computation:
    • Return 1 if the value equals M.
    • Return 0 if the value is smaller than M.

class Solution {
    /* Function to calculate mid^n compared to m:
       returns 1 if equal, 0 if less, 2 if greater */
    private int HelperFunc(int mid, int n, int m) {
        long ans = 1, baseVal = mid;
        
        while (n > 0) {
            if (n % 2 == 1) {
                ans *= baseVal;
                // Early exit if power exceeds m
                if (ans > m) return 2;
                n--;
            } else {
                n /= 2;
                baseVal *= baseVal;
                // Early exit if base squared exceeds m
                if (baseVal > m) return 2;
            }
        }
        // Check if equal
        if (ans == m) return 1;
        return 0;
    }

    // Function to find the Nth root of M
    public int NthRoot(int N, int M) {
        // Binary search on the answer space
        int low = 1, high = M;
        
        while (low <= high) {
            int mid = (low + high) / 2;
            int cmp = HelperFunc(mid, N, M);
            
            if (cmp == 1) {
                // Return mid if mid^N == M
                return mid;
            } 
            else if (cmp == 0) {
                // Move to the right half if mid^N < M
                low = mid + 1;
            } 
            else {
                // Move to the left half if mid^N > M
                high = mid - 1;
            }
        }
        // Return -1 if no nth root found
        return -1;
    }
}

package main

import "fmt"

// helperFunc calculates mid^n with early exit:
// returns 2 if mid^n > m, 1 if equal, 0 if less
func helperFunc(mid, n, m int) int {
    var ans int64 = 1
    base := int64(mid)
    exp := n

    for exp > 0 {
        if exp%2 == 1 {
            ans = ans * base
            if ans > int64(m) {
                return 2 // Early exit on overflow
            }
            exp--
        } else {
            exp /= 2
            base = base * base
            if base > int64(m) {
                return 2
            }
        }
    }
    if ans == int64(m) {
        return 1
    }
    return 0
}

// NthRoot finds the Nth root of M using binary search
func NthRoot(N int, M int) int {
    low, high := 1, M

    for low <= high {
        mid := (low + high) / 2
        midN := helperFunc(mid, N, M)

        if midN == 1 {
            // Return mid if mid^N == M
            return mid
        } else if midN == 0 {
            // Move to the right half if mid^N < M
            low = mid + 1
        } else {
            // Move to the left half if mid^N > M
            high = mid - 1
        }
    }
    // Return -1 if no nth root found
    return -1
}

func main() {
    n, m := 3, 27

    // Invoke the function
    ans := NthRoot(n, m)

    // Print the result
    fmt.Printf("The answer is: %d\n", ans)

    exponentiation by squaring method */
    func(b, exp) {
        let ans = 1;
        let base = b;

        // Exponentiation by squaring method
        while (exp > 0) {
            if (exp % 2 === 1) {
                exp--;
                ans *= base;
            } else {
                exp /= 2;
                base *= base;
            }
        }
        return ans;
    }

    /* Function to find the nth 
    root of m using linear search */
    NthRoot(N, M) {
        // Linear search on the answer space
        for (let i = 1; i <= M; i++) {
            let val = this.func(i, N);

            /* Check if the computed
            value is equal to m */
            if (val === M) {
                // Return the root value
                return i;
            } else if (val > M) {
                break;
            }
        }
        // Return -1 if no root found
        return -1;
    }
}

        return 0;
