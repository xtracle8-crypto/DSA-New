Divide two numbers without multiplication and division

Medium dsa
Bit ManipulationMath

Problem Statement

Given the two integers, dividend and divisor. Divide without using the mod, division, or multiplication operators and return the quotient.


The fractional portion of the integer division should be lost as it truncates toward zero.

As an illustration, 8.345 and -2.7335 would be reduced to 8 and -2 respectively.


Note: Assume we are dealing with an environment that could only store integers within the 32-bit signed integer range: [−231, 231 − 1]. For this problem, if the quotient is strictly greater than 231 - 1, then return 231 - 1, and if the quotient is strictly less than -231, then return -231.

Constraints

  • -231 <= dividend , divisor <= 231 - 1
  • divisor != 0

Example 1

Input : Dividend = 10 , Divisor = 3

Output : 3

Explanation : 10/3 = 3.33 , truncated to 3.

Example 2

Input : Dividend = 7 , Divisor = -3

Output : -2

Explanation : 7/-3 = -2.33 , truncated to -2.

Example 3

Input : Dividend = 7 , Divisor = 2

Editorial: Brute

Intuition:

The brute force way to solve this problem is to repeatedly add the divisor until the sum is smaller than the dividend.

Approach:

  1. Check the signs of the dividend and the divisor to determine if the result should be positive or negative. If one is positive and the other is negative, the result is negative. Otherwise, it is positive.
  2. Work with the absolute values of the dividend and the divisor to simplify the calculations.
  3. Use a loop to repeatedly add the divisor to the sum until the sum does not become greater than the dividend.
  4. Within the loop, increment the result storing the quotient and update the sum.
  5. If the result exceeds the maximum or minimum values for an integer, clamp it to the respective limit, handling the overflow cases.
  6. Adjust the sign of the result based on the initial sign determination and return it.

class Solution {
public:
    int divide(int dividend, int divisor) {
        // Base case: both numbers equal
        if (dividend == divisor) return 1;

        // Handle overflow (INT_MIN / -1)
        if (dividend == INT_MIN && divisor == -1) return INT_MAX;

        // Dividing by 1 returns the number itself
        if (divisor == 1) return dividend;

        // Determine the sign of the result
        bool isPositive = !((dividend >= 0 && divisor < 0) || (dividend < 0 && divisor > 0));

        // Convert to long long for safety and take absolute values
        long long n = abs((long long) dividend);
        long long d = abs((long long) divisor);

        // Variable to store result
        long long ans = 0;

        // Brute-force loop: repeatedly subtract divisor
        while (n >= d) {

            // Subtract divisor from dividend
            n -= d;

            // Increment quotient
            ans++;
        }

        // Check for overflow
        if (ans > INT_MAX)
            return isPositive ? INT_MAX : INT_MIN;

        // Apply sign and return
        return isPositive ? ans : -ans;
    }
};

Space Complexity: O(1) Using a couple of variables i.e., constant space.

Editorial: Optimal

Intuition:

The key idea is to repeatedly subtract the divisor from the dividend until the dividend is smaller than the divisor. However, doing this one step at a time can be very slow, so we use a method that speeds up the process by leveraging bit manipulation.

An important concept to know is that the quotient can be expressed as the sum of powers of 2.

Example: Dividend = 10, Divisor = 3.
Quotient = 10/3 = 3 which can be represented as 21 + 20.

Now, instead of subtracting the divisor from the dividend one unit at a time, we use powers of 2 (using bit shifting) to subtract larger multiples of the divisors in each step. This makes the process faster.

Approach:

  1. Check the signs of the dividend and the divisor to determine if the result should be positive or negative. If one is positive and the other is negative, the result is negative. Otherwise, it is positive.
  2. Work with the absolute values of the dividend and the divisor to simplify the calculations.
  3. Use a loop to repeatedly subtract the divisor from the dividend. Instead of subtracting the divisor one at a time, shift the divisor left (equivalent to multiplying by powers of 2) to subtract larger chunks, making the process faster.
  4. Within the loop, find the maximum power of two such that the shifted divisor is still less than or equal to the remaining part of the dividend. Subtract this value from the dividend and add the corresponding power of two to the result.
  5. If the result exceeds the maximum or minimum values for an integer, clamp it to the respective limit, handling the overflow cases
  6. Adjust the sign of the result based on the initial sign determination and return it.

class Solution {
    /* Function to divide two numbers
    without multiplication and division */
    divide(dividend, divisor) {

        // Edge cases
        if (dividend === divisor) return 1;
        if (dividend === -2147483648 && divisor === -1) return 2147483647;
        if (divisor === 1) return dividend;

        // Variable to store the sign of result
        let isPositive = true;

        // Updating the sign of quotient
        if (dividend >= 0 && divisor < 0)
            isPositive = false;
        else if (dividend <= 0 && divisor > 0)
            isPositive = false;

        // Storing absolute dividend & divisor (as BigInt)
        let n = BigInt(Math.abs(dividend));
        let d = BigInt(Math.abs(divisor));

        // Variable to store the answer
        let ans = 0n;

        /* Looping while dividend is 
        greater than equal to divisor */
        while (n >= d) {
            let count = 0n;
            let curr = d;

            /* Finding the required 
            largest power of 2 */
            while (n >= (curr << 1n)) {
                curr <<= 1n;
                count += 1n;
            }

            // Updating the answer & dividend
            ans += (1n << count);
            n -= curr;
        }

        // Convert back to Number and handle overflowing condition
        let out = Number(ans);
        if (isPositive && out === 2147483648) return 2147483647;
        if (!isPositive && out === 2147483648) return -2147483648;

        /* Returning the quotient 
        with proper sign */
        return isPositive ? out : -out;
    }
}

package main

import (
    "fmt"
    "math"
)

/* Function to divide two numbers
without multiplication and division */
func divide(dividend int, divisor int) int {
    // Edge cases
    if divisor == 0 {
        return math.MaxInt32
    }
    if dividend == divisor {
        return 1
    }
    if dividend == math.MinInt32 && divisor == -1 {
        return math.MaxInt32
    }

    // Variable to store the sign of result
    positive := (dividend < 0) == (divisor < 0)

    // Storing absolute dividend & divisor using int64
    a := int64(dividend)
    b := int64(divisor)
    if a < 0 {
        a = -a
    }
    if b < 0 {
        b = -b
    }

    var result int64 = 0

    // Loop to subtract large multiples of divisor using bit shifts
    for a >= b {
        temp := b
        multiple := int64(1)
        for temp<<1 <= a {
            temp <<= 1
            multiple <<= 1
        }
        a -= temp
        result += multiple
    }

    // Apply sign
    if !positive {
        result = -result
    }

    // Clamp to 32-bit signed integer limits
    if result > math.MaxInt32 {
        return math.MaxInt32
    }
    if result < math.MinInt32 {
        return math.MinInt32
    }
    return int(result)
}

func main() {
    dividend := 10
    divisor := 3

    /* Function call to divide two numbers
    without multiplication and division */
    ans := divide(dividend, divisor)

    fmt.Printf("The result of dividing %d and %d is %d\n", dividend, divisor, ans)
}

func divide(dividend int, divisor int) int {
    // Handle overflow case
    if dividend == math.MinInt32 && divisor == -1 {
        return math.MaxInt32
    }

    // Determine the sign of the result
    positive := (dividend < 0) == (divisor < 0)

    // Work with absolute values using int64 to avoid overflow
    dvd := int64(dividend)
    dvr := int64(divisor)
    if dvd < 0 {
        dvd = -dvd
    }
    if dvr < 0 {
        dvr = -dvr
    }

    var ans int64 = 0
    // Perform division using bit shifts (subtraction of multiples of divisor)
    for shift := 31; shift >= 0; shift-- {
        if (dvr << shift) <= dvd {
            dvd -= dvr << shift
            ans += 1 << shift
        }
    }

    // Apply the correct sign
    if !positive {
        ans = -ans
    }

    // Clamp to 32-bit signed integer limits if needed
    if ans > math.MaxInt32 {
        return math.MaxInt32
    }
    if ans < math.MinInt32 {
        return math.MinInt32
    }
    return int(ans)
}

            return 2**31 - 1

        # Dividing by 1 returns the number itself
        if divisor == 1:
            return dividend

        # Determine the sign of the result
        isPositive = not ((dividend >= 0 and divisor < 0) or
                          (dividend < 0 and divisor > 0))

        # Take absolute values
        n = abs(dividend)
        d = abs(divisor)

        # Variable to store result
        ans = 0

        # Brute-force loop: repeatedly subtract divisor
        while n >= d:

            # Subtract divisor from dividend
            n -= d

            # Increment quotient
            ans += 1

        # Check for overflow
        if ans > 2**31 - 1:
            return (2**31 - 1) if isPositive else -2**31

        # Apply sign and return
        return ans if isPositive else -ans

func divide(dividend int, divisor int) int {
    // Edge cases
    if divisor == 0 {
        return math.MaxInt32
    }
    if dividend == divisor {
        return 1
    }
    if dividend == math.MinInt32 && divisor == -1 {
        return math.MaxInt32
    }

    // Variable to store the sign of result
    positive := (dividend < 0) == (divisor < 0)

    // Storing absolute dividend & divisor using int64
    a := int64(dividend)
    b := int64(divisor)
    if a < 0 {
        a = -a
    }
    if b < 0 {
        b = -b
    }

    var result int64 = 0

    // Loop to subtract large multiples of divisor using bit shifts
    for a >= b {
        temp := b
        multiple := int64(1)
        for temp<<1 <= a {
            temp <<= 1
            multiple <<= 1
        }
        a -= temp
        result += multiple
    }

    // Apply sign
    if !positive {
        result = -result
    }

    // Clamp to 32-bit signed integer limits
    if result > math.MaxInt32 {
        return math.MaxInt32
    }
    if result < math.MinInt32 {
        return math.MinInt32
    }
    return int(result)
}

        let isPositive = true;

        // Updating the sign of quotient
        if (dividend >= 0 && divisor < 0)
            isPositive = false;
        else if (dividend <= 0 && divisor > 0)
            isPositive = false;

        // Storing absolute dividend & divisor (as BigInt)
        let n = BigInt(Math.abs(dividend));
        let d = BigInt(Math.abs(divisor));

        // Variable to store the answer
        let ans = 0n;

        /* Looping while dividend is 
        greater than equal to divisor */
        while (n >= d) {
            let count = 0n;
            let curr = d;

            /* Finding the required 
            largest power of 2 */
            while (n >= (curr << 1n)) {
                curr <<= 1n;
                count += 1n;

            // Updating the answer & dividend
            ans += (1n << count);
            n -= curr;
        }

        // Convert back to Number and handle overflowing condition
        let out = Number(ans);
        if (isPositive && out === 2147483648) return 2147483647;
        if (!isPositive && out === 2147483648) return -2147483648;

        /* Returning the quotient 
        with proper sign */
        return isPositive ? out : -out;
    }
