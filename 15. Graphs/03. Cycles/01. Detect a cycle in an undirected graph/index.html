Detect a cycle in an undirected graph

Hard dsa
GraphDepth-First SearchUnion Find

Problem Statement

Given an undirected graph with V vertices labeled from 0 to V-1. The graph is represented using an adjacency list where adj[i] lists all nodes connected to node. Determine if the graph contains any cycles.

Note: The graph does not contain any self-edges (edges where a vertex is connected to itself).

Constraints

  • E=number of edges
  • 1 ≤ V, E ≤ 104

Example 1


Input: V = 6, adj= [[1, 3], [0, 2, 4], [1, 5], [0, 4], [1, 3, 5], [2, 4]]

Output: True

Explanation: The graph contains a cycle: 0 ->1 -> 2 -> 5 -> 4 -> 1.

Example 2

Input: V = 4, adj= [[1, 2], [0], [0, 3], [2]]

Output: False

Explanation: The graph does not contain any cycles.

Example 3

Input: V = 4, adj= [[1, 2], [0, 2], [0, 1, 3], [2]]

Fun Fact

Fact: Detecting cycles in a graph is a fundamental problem that has a wide array of applications in the software industry. It is used in networking and telecommunication for finding routing loops or cyclical data dependencies. Also, it finds usage in DevOps for spotting potential infinite loops in tasks automation like ansible, puppet, etc. More so, in social networking applications like Facebook or LinkedIn, it can help track circular friend requests or connections. Algorithms for detecting cycles are extensively used in garbage collection mechanism in programming languages like Python, Java and Javascript to identify and clean up objects that are no longer needed by the program to efficiently manage memory.

Editorial: BFS

Intuition:

In an undirected graph, a cycle is formed when a path exists that returns to the starting vertex without reusing an edge. The key idea is that during traversal (e.g., using Breadth-First Search (BFS) for this approach), if we encounter a vertex that has already been visited and is not the immediate parent of the current vertex, a cycle exists.

Approach:

  1. A helper function performs BFS to detect cycles, using a queue to track nodes and their parents. For each unvisited node, initialize the queue with the node and mark it as visited.
  2. While the queue is not empty, process each node's neighbors:
    • If a neighbor is unvisited, mark it visited and enqueue it.
    • If a neighbor is visited and not the parent, a cycle is detected.
  3. Return true if any cycle is found during traversal.

class Solution {
    
    // Function to perform BFS traversal
    private boolean bfs(int i, 
        List adj[], 
        boolean[] visited) {
            
        // Queue to store {node, parent}
        Queue q = new LinkedList<>();
        
        /* Push initial node in queue 
        with no one as parent */
        q.add(new int[]{i, -1});
        
        // Mark the node as visited
        visited[i] = true;
        
        // Until the queue is empty
        while (!q.isEmpty()) {
            
            // Get the node and its parent
            int[] current = q.poll();
            int node = current[0];
            int parent = current[1];
            
            // Traverse all the neighbors
            for (int it : adj[node]) {
                
                // If not visited
                if (!visited[it]) {
                    // Mark the node as visited
                    visited[it] = true;
                    
                    /* Push the new node in queue 
                    with curr node as parent */
                    q.add(new int[]{it, node});
                } 
                
                /* Else if it is visited with some 
                different parent a cycle is detected */
                else if (it != parent) return true;
            }
        }
        return false;
    }

        boolean[] visited = new boolean[V];

                // Start BFS traversal and update result
                ans = bfs(i, adj, visited);
                
                // Break if a cycle is detected
                if (ans) break;

public class Solution {

    // Function to perform BFS traversal
    private bool bfs(int i, List[] adj, bool[] visited) {
        // Queue to store {node, parent}
        Queue> q = new Queue>();
        
        // Push initial node in queue with no one as parent
        q.Enqueue(Tuple.Create(i, -1));
        
        // Mark the node as visited
        visited[i] = true;
        
        // Until the queue is empty
        while (q.Count > 0) {
            // Get the node and its parent
            Tuple current = q.Dequeue();
            int node = current.Item1;
            int parent = current.Item2;
            
            // Traverse all the neighbors
            foreach (int neighbor in adj[node]) {
                // If not visited
                if (!visited[neighbor]) {
                    // Mark the node as visited
                    visited[neighbor] = true;
                    
                    /* Push the new node in queue 
                    with curr node as parent */
                    q.Enqueue(Tuple.Create(neighbor, node));
                } 
                /* Else if it is visited with some 
                different parent a cycle is detected */
                else if (neighbor != parent) {
                    return true;
                }
            }
        }
        return false;
    }

    // Function to detect cycle in an undirected graph.
    public bool IsCycle(int V, List[] adj) {
        bool[] visited = new bool[V]; // Default initialized to false
        bool ans = false;
        
        // Start Traversal from every unvisited node
        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                // Start BFS traversal and update result
                ans = bfs(i, adj, visited);
                
                // Break if a cycle is detected
                if (ans) break;
            }
        }
        return ans;
    }
}

package main

import "fmt"

// Function to perform BFS traversal
func bfs(i int, adj [][]int, visited []bool) bool {
	// Queue to store {node, parent}
	q := []struct {
		node   int
		parent int
	}{}

	// Push initial node in queue with no one as parent
	q = append(q, struct {
		node   int
		parent int
	}{i, -1})

	// Mark the node as visited
	visited[i] = true

	// Until the queue is empty
	for len(q) > 0 {
		// Get the node and its parent
		node, parent := q[0].node, q[0].parent
		q = q[1:]

		// Traverse all the neighbors
		for _, it := range adj[node] {
			// If not visited
			if !visited[it] {
				// Mark the node as visited
				visited[it] = true

				// Push the new node in queue with curr node as parent
				q = append(q, struct {
					node   int
					parent int
				}{it, node})
			} else if it != parent {
				// Else if it is visited with some different parent a cycle is detected
				return true
			}
		}
	}
	return false
}

// Function to detect cycle in an undirected graph.
func isCycle(V int, adj [][]int) bool {
	// Visited array
	visited := make([]bool, V)

	// Variable to store if there is a cycle detected
	var ans bool

	// Start Traversal from every unvisited node
	for i := 0; i < V; i++ {
		if !visited[i] {
			// Start BFS traversal and update result
			ans = bfs(i, adj, visited)

			// Break if a cycle is detected
			if ans {
				break
			}
		}
	}
	return ans
}

Intuition:

In an undirected graph, a cycle is formed when a path exists that returns to the starting vertex without reusing an edge. The key idea is that during traversal (e.g., using Depth-First Search (DFS) for this approach), if we encounter a vertex that has already been visited and is not the immediate parent of the current vertex, a cycle exists.

Approach:

  1. A helper function performs DFS to detect cycles, keeping track of nodes and their parents. For each unvisited node, initialize the mark the node as visited.
  2. Traverse the neighbors of the node:
    • If a neighbor is unvisited, mark it visited and recursively perform DFS from it.
    • If a neighbor is visited and not the parent, a cycle is detected.
  3. Return true if any cycle is found during traversal.

    bool bfs(int i, vector<int> adj[],

        queue<pair<int, int>> q;
        
        /* Push initial node in queue 
        with no one as parent */
        q.push({i, -1});
        
        // Mark the node as visited
        visited[i] = true;
        
        // Until the queue is empty
        while(!q.empty()) {
            
            // Get the node and its parent
            int node = q.front().first;
            int parent = q.front().second;

            for(auto it : adj[node]) {

                if(!visited[it]) {

func bfs(i int, adj [][]int, visited []bool) bool {
	// Queue to store {node, parent}
	q := []struct {
		node   int
		parent int
	}{}

	// Push initial node in queue with no one as parent
	q = append(q, struct {
		node   int
		parent int
	}{i, -1})

	// Mark the node as visited
	visited[i] = true

	// Until the queue is empty
	for len(q) > 0 {
		// Get the node and its parent
		node, parent := q[0].node, q[0].parent
		q = q[1:]

		// Traverse all the neighbors
		for _, it := range adj[node] {
			// If not visited
			if !visited[it] {
				// Mark the node as visited
				visited[it] = true

				// Push the new node in queue with curr node as parent

					node   int
					parent int
				}{it, node})
			} else if it != parent {
				// Else if it is visited with some different parent a cycle is detected
				return true
			}
		}
	}
	return false
}
